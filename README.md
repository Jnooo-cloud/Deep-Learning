Dieses Repository reproduziert die Kursaufgabe "Deep Learning" aus dem Udemy-Kurs "Python fuer Data Science, Machine Learning & Visualization".

## Inhalt

- `5-Tensorflow_Projekt_new.ipynb`: Jupyter-Notebook mit der Loesung.
- `bank_note_data.csv`: Datensatz zur Klassifikation echter und gefaelschter Banknoten.
- `requirements.txt`: Python-Abhaengigkeiten fuer Binder.

## Ausfuehrung

1. Klicke auf den Binder- oder Colab-Badge.
2. Oeffne das Notebook `5-Tensorflow_Projekt_new.ipynb`, falls es nicht direkt geoeffnet wird.
3. Fuehre alle Zellen in der vorgegebenen Reihenfolge aus.

## Erwartetes Ergebnis

Das Notebook skaliert die Merkmale des Datensatzes, trainiert ein TensorFlow-Modell zur binaren Klassifikation und erzeugt danach Vorhersagen fuer die Testdaten. Als Ergebnis erscheinen insbesondere eine `confusion_matrix` und ein `classification_report`, mit denen die Modellqualitaet fuer die Erkennung echter und gefaelschter Banknoten nachvollzogen werden kann.
